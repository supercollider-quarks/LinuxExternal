BusMonitor class to monitor a control bus graphically with GNUPlot


Inherits from: Object


This creates a GNUPlot.monitor on the bus, with an update time of 0.05, a data size of 1000, and a skip of 20.


See also: GNUPlot


You need to have GNUPlot installed in order to be able to use this class!


Creation / Class Methods


*new( b )

Create a monitor on bus "b".


Accessing Instance and Class Variables

setRange(min=0.0,max=1.0)

Set the minimum and maximum of the plotting range.

start

Start the monitor.

stop

Stop the monitor.

dt_

Set the update time of the monitor (default 0.05).

cleanUp

Quit GNUPlot and close the pipe to GNUPlot.

Example:

s = Server.local.boot;

b = Bus.control(s,1);

{ Out.kr( b.index, Mix.new( SinOsc.kr( [0.125, 0.25], 0, [0.6,0.4] ) ) ) }.play;

a = BusMonitor.new( b );

a.start;

a.setRange( -1, 1 );

a.stop;

a.cleanUp;
